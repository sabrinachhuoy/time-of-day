Greeting

This purpose of this webpage is it indicate the current hour using Javascript. Depending on the hour this webpage will present a different image, message, and color.

I forked the repo from our professor and cloned it to my local machine. We determined that in order to get a date object to pop up an alert we must use an if block. Prior to this we used the Eval Form from Module 8 to test this if block.

In addition, we added a console.log at the end of the if block to track any errors when we inspect the browser.

We added "let myColor = " to add a background color to the HTML tag.

In order to have pictures we added a src. attribute which displays a unique picture.

Its very important that we add notes to keep us on track. Throughout the code there are hidden notes that tells the story of the code.

We determined the hour by adding the myHour variable so there is a else if block for Morning, Afternoon, Evening, and Night
